# Boston Housing Price Prediction 🏠

A simple linear regression project built from scratch using Gradient Descent to predict Boston house prices.

## 🚀 Key Features
* **Custom Implementation:** Linear regression and gradient descent built from scratch—no `sklearn` for the algorithm itself.
* **Feature Engineering:** Analysis of multicollinearity to balance model accuracy and simplicity.
* **Optimization:** Data normalized using Z-score scaling to ensure stable convergence.

## 📈 Results
* **Method:** Linear Regression (Gradient Descent)
* **Accuracy:** ~60% R-squared

## 📂 Structure
* `Boston_Housing_Regression.ipynb`: Code, analysis, and visualization.
* `Boston.csv`: The dataset used.
